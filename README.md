# 🕹️ Jogo da Forca em Python

Este projeto é uma implementação simples do clássico jogo da forca utilizando Python no terminal.

## 🎯 Objetivo

O principal objetivo deste projeto foi praticar e consolidar conceitos fundamentais de programação, como:

* Controle de fluxo (`while`, `if/else`)
* Manipulação de listas
* Funções e organização de código
* Validação de entrada do usuário
* Gerenciamento de estado ao longo da execução

## ⚙️ Funcionalidades

* Palavra aleatória escolhida a partir de uma lista
* Sistema de tentativas limitadas
* Validação de entrada (apenas uma letra válida)
* Controle de letras já utilizadas
* Atualização dinâmica da palavra oculta
* Mensagens de vitória e derrota

## 🧠 Lógica do Projeto

O jogo mantém um estado interno da palavra oculta, atualizando as posições corretas conforme o usuário acerta letras.

Também controla:

* Número de tentativas restantes
* Letras já utilizadas
* Condição de vitória (quando não há mais "-" na palavra)

## ▶️ Como executar

1. Certifique-se de ter Python instalado
2. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/seu-repo.git
```
